## 关键开发规范（来自 impl-plan UAT 总结）

1. **翻译文字**：所有 UI 文字必须用 `t_translation` 数据库 + `useLocaleTranslation()` / `isReadServerTextData`，不能硬编码
2. **图表颜色**：统一用 `getProgressColor()`：≤70% 红 `#F5626C`，71-89% 橙 `#F78956`，≥90% 绿 `#45AD5E`
3. **文件上传参数**：FormData key 是 `'file'`（单数），不是 `'files'`
4. **防抖 hook**：debounce 函数放在 `useCallback` 里，防止每次渲染创建新引用
5. **dashboard chart**：新增 CHAR_* 必须同时在 DashBoardChartsEdit 和 DashBoardChartsShow 两个文件都加 case 分支

**How to apply:** 开始任何 V3.3 未完成需求前先查此表，确认文件路径、API 路由和翻译键。
